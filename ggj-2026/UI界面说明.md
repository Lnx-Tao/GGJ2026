# 舞者面具界面说明

## 🎨 界面布局

界面会在屏幕中央显示，包含以下元素：

### 1. **顶部指令文本**
- 位置：屏幕上方中央
- 内容："舞者应随着旋律舞动"
- 样式：白色，32号字体

### 2. **节奏轨道（滑动条）**
- 位置：屏幕中央水平线（Y=400）
- 样式：白色线条，宽度3像素
- 长度：600像素（从X=100到X=700）

### 3. **节奏点（圆圈）**
- 位置：在轨道上均匀分布
- 数量：默认4个（可配置）
- 样式：紫色方块（20x20像素），代表圆圈
- 颜色：初始为紫色 `Color(0.6, 0.2, 0.8)`
- 成功时：变为绿色 `Color(0.2, 0.8, 0.2)`

### 4. **移动指针**
- 位置：轨道上方，从左侧开始
- 样式：红色垂直条（4像素宽，60像素高）
- 移动：从左向右匀速移动
- 速度：默认200像素/秒（可配置）

### 5. **分数显示**
- 位置：屏幕下方中央
- 内容："舞姿正确: X/Y"（X是已击中数，Y是总数）
- 样式：白色，24号字体
- 更新：实时更新

## 🎮 界面效果示意

```
┌─────────────────────────────────────────┐
│                                         │
│        舞者应随着旋律舞动                │
│                                         │
│                                         │
│    ●     ●     ●     ●                 │
│    ────────────────────────             │
│    │                                    │
│                                         │
│        舞姿正确: 0/4                    │
│                                         │
└─────────────────────────────────────────┘
```

说明：
- `●` = 紫色节奏点（圆圈）
- `─` = 白色轨道线
- `│` = 红色移动指针（从左侧开始向右移动）

## 🖱️ 操作方式

1. 游戏开始后，红色指针从左侧开始向右移动
2. 当指针进入紫色圆圈范围内时
3. 点击鼠标左键或右键进行判定
4. 成功击中的圆圈会变成绿色
5. 需要击中所有节奏点才能成功

## 🧪 测试方法

运行游戏后，**按空格键**即可触发舞者面具界面进行测试！

## 📐 坐标位置

- 轨道起点：X=100, Y=400
- 轨道终点：X=700, Y=400
- 指令文本：Y=200（居中）
- 分数文本：Y=500（居中）
- 指针初始位置：X=98, Y=370（高度60像素）

## 🎨 颜色方案

- 轨道：白色 `Color(1, 1, 1, 1)`
- 节奏点（初始）：紫色 `Color(0.6, 0.2, 0.8, 1.0)`
- 节奏点（成功）：绿色 `Color(0.2, 0.8, 0.2, 1.0)`
- 指针：红色 `Color(1, 0, 0, 1)`
- 文本：白色 `Color(1, 1, 1, 1)`

## 💡 提示

界面会全屏覆盖（Control节点铺满整个屏幕），但不会阻挡游戏背景的显示（如果需要的话可以添加半透明背景）。
