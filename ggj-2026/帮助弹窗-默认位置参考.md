## 🎯 位置计算说明

### 屏幕尺寸：1600 x 900

```
       0,0 ────────────────────────── 1600,0
        │                              │
        │         800,450              │
        │           (中心)             │
        │                              │
     0,900 ──────────────────────── 1600,900
```

### 元素实际屏幕位置

以 **HelpImage** 为例：
```
锚点: 0.5, 0.5 = 800, 450（屏幕中心）
Offset: left: -500, top: -300, right: 500, bottom: 300

实际屏幕坐标:
左上角: 800 + (-500), 450 + (-300) = (300, 150)
右下角: 800 + 500, 450 + 300 = (1300, 750)

结果：图片显示在 (300, 150) 到 (1300, 750)
尺寸：1000 x 600 像素
```

NextButton:  在 (970, 830) 到 (1120, 880)  [右下区域]

## 🔧 常用调整速查

### 快速调整图片大小

| 目标尺寸 | Left | Top | Right | Bottom |
|----------|------|-----|-------|--------|
| 800x500 | -400 | -250 | 400 | 250 |
| 1000x600 | -500 | -300 | 500 | 300 | ← 默认
| 1200x700 | -600 | -350 | 600 | 350 |
| 1400x800 | -700 | -400 | 700 | 400 |

### 按钮垂直位置调整

**向上移动**（数值减小）：
```
top: 380 → 300     # 向上移 80px
bottom: 430 → 350
```

**向下移动**（数值增大）：
```
top: 380 → 420     # 向下移 40px
bottom: 430 → 470
```

### 按钮水平间距调整

**间距缩小**：
```
PrevButton.right: -170 → -120   # 向右移
CloseButton.left: -60 → -80     # 向左移
CloseButton.right: 60 → 80      # 向右移
NextButton.left: 170 → 120      # 向左移
```

**间距增大**：反向操作

## 📐 对齐参考线

### 垂直中线（Left-Right 对称）
```
Left + Right = 0
示例：left: -50, right: 50  ✓ 对称
     left: -60, right: 40  ✗ 不对称
```

### 水平中线（Top-Bottom 对称）
```
Top + Bottom = 0
示例：top: -100, bottom: 100  ✓ 对称
     top: -100, bottom: 80   ✗ 不对称
```

### 元素对齐
```
左对齐：所有元素的 left 值相同
右对齐：所有元素的 right 值相同
顶部对齐：所有元素的 top 值相同
底部对齐：所有元素的 bottom 值相同
```

## 💡 调整技巧

### 保持元素尺寸不变
```
宽度 = right - left
高度 = bottom - top

移动时保持差值不变：
left: -100, right: 100  (宽度 200)
移到右侧：
left: 200, right: 400   (宽度仍为 200)
```

### 调整元素尺寸
```
放大：增加 right-left 和 bottom-top 的差值
缩小：减小 right-left 和 bottom-top 的差值

示例（等比放大 1.2 倍）：
原：left: -100, right: 100 (200px)
新：left: -120, right: 120 (240px)
```

### 批量调整
在 Godot 中选中多个节点（Ctrl+点击），可以一起移动。

## ⚠️ 注意事项

1. **负数方向**：
   - left 负值 = 向左
   - top 负值 = 向上
   - right 正值 = 向右
   - bottom 正值 = 向下

2. **锚点影响**：
   - 修改锚点会改变参考点
   - 建议同类元素使用相同锚点

3. **分辨率适配**：
   - 在 800x600 和 1920x1080 下都要测试
   - 使用百分比锚点更具适应性
