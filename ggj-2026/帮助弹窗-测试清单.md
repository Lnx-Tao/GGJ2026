## 🐛 常见问题检查

### 图片无法显示
- [ ] 检查 `Art` 文件夹中是否存在 help_img_1.png ~ help_img_7.png
- [ ] 在 Godot 编辑器中查看图片是否正确导入
- [ ] 检查控制台是否有错误信息

### 弹窗无法打开
- [ ] 检查 `Scene/HelpPopup.tscn` 文件是否存在
- [ ] 检查 `Script/help_popup.gd` 文件是否存在
- [ ] 查看控制台错误信息

### 按钮无响应
- [ ] 检查按钮是否正确连接信号
- [ ] 确认按钮可见且未被遮挡
- [ ] 检查鼠标是否正常工作
