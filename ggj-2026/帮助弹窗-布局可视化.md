## 🎨 不同布局方案可视化

### 方案 A：默认布局（推荐）

```
┌──────────────────────────┐
│                          │
│    ┌──────────────┐      │
│    │   大图展示   │      │
│    └──────────────┘      │
│         1 / 7            │
│   [◀]  [×]  [▶]         │
└──────────────────────────┘
```

**特点**：
- 图片居中显著
- 控制按钮底部横排
- 易于操作

---

### 方案 B：全屏沉浸式

```
┌──────────────────────────┐ ×
│                          │
│                          │
│     全屏超大图片         │
│                          │
│                          │
└──────────────────────────┘
         ◀    ▶
      图外左右箭头
```

**特点**：
- 图片占满大部分空间
- 控制按钮在图片外侧
- 沉浸式体验

**调整**：
```
HelpImage: left: -700, top: -420, right: 700, bottom: 420
PrevButton: left: -750, top: -25, right: -700, bottom: 25
NextButton: left: 700, top: -25, right: 750, bottom: 25
CloseButton: 移到右上角（锚点改为 1.0, 0.0）
PageLabel: 移到图片内（left: 550, top: 370）
```

---

### 方案 C：侧边栏控制

```
┌──────────────────────┬──┐
│                      │◀ │
│                      │  │
│    图片展示          │× │
│                      │  │
│                      │▶ │
│       1 / 7          │  │

**特点**：

CloseButton: left: -140, top: -25, right: -20, bottom: 25
NextButton: left: -140, top: 50, right: -20, bottom: 100
```

## 📏 标准尺寸参考

### 按钮标准尺寸

| 按钮类型 | 宽度 | 高度 | 用途 |
|----------|------|------|------|
| 小按钮 | 80px | 40px | 图标按钮 |
| 标准按钮 | 120px | 50px | 文字按钮 |
| 大按钮 | 150px | 50px | 当前默认 |
| 超大按钮 | 200px | 60px | 强调操作 |

### 间距标准

| 间距类型 | 尺寸 | 说明 |
|----------|------|------|
| 紧密 | 10-20px | 相关元素 |
| 标准 | 30-50px | 一般间距 |
| 宽松 | 80-120px | 分组间距 |
| 当前 | 110px | 按钮之间 |

## 🎯 调整示例

### 示例 1：让图片更大

**目标**：从 1000x600 放大到 1200x700

```
修改前：
offset_left: -500
offset_top: -300
offset_right: 500
offset_bottom: 300

修改后：
offset_left: -600   (增加 100)
offset_top: -350    (增加 50)
offset_right: 600   (增加 100)
offset_bottom: 350  (增加 50)
```

### 示例 2：让按钮更靠近图片

**目标**：按钮上移 50px

```
修改前：
top: 380
bottom: 430

修改后：
top: 330    (减少 50)
bottom: 380 (减少 50)
```

### 示例 3：增加按钮间距

**目标**：按钮之间距离从 110px 增加到 150px

```
PrevButton:
  right: -170 → -190  (向左移 20)

CloseButton:
  保持不变

NextButton:
  left: 170 → 190     (向右移 20)
```

## 💡 可视化调试技巧

### 在 Godot 编辑器中

1. **显示网格**：视图 → 显示网格
2. **显示标尺**：可以看到像素坐标
3. **预览不同分辨率**：
   - 项目设置 → 显示 → 窗口
   - 修改测试窗口尺寸

### 实时预览

1. 修改 offset 值
2. 在编辑器中立即看到变化
3. 按 F5 运行游戏测试实际效果
